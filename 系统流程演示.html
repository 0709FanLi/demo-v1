📊 系统简介

这是一个基于人工智能的抗衰老专家咨询系统，就像随身携带的健康顾问，可以随时回答您关于抗衰老、健康管理的专业问题。

🎯 系统能做什么？

  • 💬 智能对话：像和专家聊天一样，提出任何抗衰老相关问题
  • 📚 知识库支持：基于科学文献和研究成果，给出专业可靠的建议
  • 📊 图片分析：上传体检报告或身体指标图片，获得专业解读
  • 🎯 个性化建议：根据您的具体情况，提供定制化的健康方案
  • 🔍 智能过滤：自动识别问题是否在专业范围内，避免误导

🎨 前端界面

美观易用的聊天界面，支持文字和图片输入，就像使用微信一样简单

🧠 AI 大模型

使用阿里云通义千问，具备强大的理解和回答能力

📚 知识库

存储专业的抗衰老知识，确保回答的科学性和准确性

🔍 智能检索

快速找到最相关的知识，提高回答质量

👤 用户使用流程（从提问到获得答案）

1️⃣ 用户提问

在聊天界面输入问题，例如：

  • "NMN 对抗衰老有什么作用？"
  • "如何通过饮食延缓衰老？"
  • "端粒与细胞衰老的关系？"

也可以上传体检报告图片

⬇️

2️⃣ 系统接收

系统收到问题后，立即开始处理：

  • 显示"正在思考..."动画
  • 将问题发送到后端服务器
  • 准备调用知识库和 AI
⬇️

3️⃣ 知识库检索

系统在知识库中搜索相关内容：

  • 将问题转换为向量（数字表示）
  • 在知识库中找最相似的内容
  • 计算相关性评分（0-1分）
  • 判断是否在专业范围内
⬇️

4️⃣ 智能判断

关键步骤：判断问题是否在专业范围内

  • ✅ 相关性 ≥ 0.60：继续回答
  • ❌ 相关性 < 0.60：提示超出范围

例如："今天天气怎么样？"会被识别为不相关

⬇️

5️⃣ AI 生成答案

如果问题相关，AI 开始生成答案：

  • 结合知识库的专业内容
  • 考虑历史对话上下文
  • 生成专业、易懂的回答
  • 标注置信度（高/中/低）
⬇️

6️⃣ 返回结果

用户在界面上看到答案：

  • 📝 完整的回答内容
  • 🎯 置信度标签（高/中/低）
  • 📚 引用的知识来源
  • ⏰ 回答时间戳

💡 特殊情况处理

如果问题超出专业范围：
系统会友好地告知用户："抱歉，您的问题似乎超出了我的专业知识库范围。我目前专注于抗衰老、健康管理等领域..." 并建议用户咨询相关领域的专业人士。

⚙️ 技术实现流程（系统内部如何工作）

🌐 前端（用户界面）

技术：React + TypeScript

  • 美观的聊天界面
  • 实时消息显示
  • 图片上传预览
  • 知识库管理界面
→

🔌 API 接口

前后端通信桥梁

  • POST /api/v1/chat/ - 对话
  • GET /api/v1/knowledge/list - 获取知识
  • POST /api/v1/knowledge/add - 添加知识
⬇️

🖥️ 后端服务器

技术：Python + FastAPI

  • 接收用户请求
  • 协调各个服务
  • 处理业务逻辑
  • 返回处理结果
⬇️

🔍 向量检索

技术：Sentence Transformers

  • 将文本转为向量（数字）
  • 计算语义相似度
  • 找到最相关的知识
  • 评估相关性评分
→

💾 向量数据库

技术：Milvus

  • 存储知识向量
  • 高速相似度搜索
  • 支持海量数据
  • 毫秒级响应
⬇️

🧠 AI 大模型

技术：阿里云通义千问

  • 理解用户问题
  • 结合知识库内容
  • 生成专业回答
  • 支持图文理解
⬇️

✅ 质量控制

确保回答质量

  • 相似度阈值判断（0.60）
  • 置信度评估
  • 超出范围提示
  • 日志记录追踪

🔧 关键技术说明

1. 向量化技术：将文字转换为计算机能理解的数字，就像给每个句子一个"指纹"

2. 相似度计算：比较两个"指纹"的相似程度，找到最匹配的知识

3. RAG 技术：检索增强生成，让 AI 基于真实知识回答，而不是"瞎编"

💡 使用示例

示例 1：询问营养补充

用户："NMN 对抗衰老有什么作用？"
系统：会检索知识库，找到关于 NMN 的专业资料，然后结合 AI 生成详细的回答，包括作用机制、推荐剂量、注意事项等。

示例 2：超出范围的问题

用户："今天天气怎么样？"
系统：会识别出这个问题与抗衰老无关，友好地提示："抱歉，您的问题似乎超出了我的专业知识库范围..."

示例 3：上传图片分析

用户：上传体检报告图片，问"我的血脂指标正常吗？"
系统：会识别图片中的数据，结合知识库给出专业解读和改善建议。

🎯 管理知识库

查看知识：点击顶部"📚 知识库: X 条"，可以查看所有已添加的知识

添加知识：点击"➕ 添加知识"按钮，输入新的专业知识和分类

删除知识：在知识列表中，点击"🗑️ 删除"按钮可以移除过时的内容

刷新列表：点击"🔄 刷新"按钮更新知识库显示
